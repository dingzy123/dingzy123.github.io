  • 极简三入口：写代码 / 看资料 / 练习

工业级技术底座

基于 Ubuntu 24.04 LTS，完整兼容性 + 安全只读架构

系统核心

  • Lubuntu 24.04 LTS (Noble)
  • 内核 Linux 6.8 + LXQt 轻量桌面
  • UEFI / Legacy 双模式启动
  • casper-rw 持久化 + overlayroot

编译工具链

  • g++ 13.2.0 (完整 libstdc++)
  • 默认 C++14 / 支持 C++17
  • Code::Blocks 20.03 + Geany
  • GDB 14.1 调试集成

安全持久化

  • 根文件系统 squashfs 只读
  • 用户数据 ext4 独立分区
  • OverlayFS 透明读写
  • 防误删 / 一键重置

离线判题系统

基于 Project LemonLime 引擎，完美模拟 NOIP / CSP 复赛环境。无需网络，提交即评测。

  • 支持传统题 / 交互题 / 通信题 / 提交答案题
  • 多测试点 + 子任务 + 时间/内存限制检测
  • 文件 IO (freopen) 自动评测，返回 AC/WA/TLE/MLE
